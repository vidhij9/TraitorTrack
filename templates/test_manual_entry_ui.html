This page tests the manual parent entry endpoint directly.

Valid examples: SB12345, sb00860, Sb00736, sB00123
Invalid examples: SB123 (too short), 12345 (missing SB prefix), sb12345a (extra character)

How This Works:

  1. Enter a Bill ID (must be a valid bill in the database)
  2. Enter a Parent Bag QR code in format SB##### (e.g., SB12345)
  3. Click "Test Manual Entry" to send a POST request
  4. The system will validate and process the request
Note: You must be logged in as an Admin or Biller to use this feature.
