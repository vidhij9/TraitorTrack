About Locations

Locations are the physical places where bags are scanned. They help you track where each bag has been throughout its journey:

  • Warehouses
  • Distribution centers
  • Retail locations
  • Processing facilities
  • Other checkpoints

You need to select a location before starting the bag scanning process.

Location Tips

For effective location tracking:

  • Use consistent naming conventions
  • Include address information when available
  • Create separate locations for different areas even within the same facility (e.g., Warehouse A - Receiving, Warehouse A - Shipping)
  • Select the correct location before starting a scanning session
