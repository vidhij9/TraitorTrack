‚ùå Why You're Getting a 400 Error

The URL /bill/manual_parent_entry is an API endpoint that only accepts POST requests with form data. When you navigate to it directly in your browser, you're sending a GET request, which causes the error.

‚úÖ How to Use Manual Parent Entry Correctly

Manual parent entry should be done from the Scan Parent Bags page:

Correct Steps:

Step 1: Go to Bill Management

Navigate to: /bills or /bill_management

Step 2: Create or Select a Bill

Click on a bill to scan parent bags, or create a new one

Step 3: Use the Manual Entry Form

On the scanning page, you'll see a "Manual Entry" section with:

  • An input field labeled "Enter parent bag QR (SB12345)"
  • An "Add" button to submit the manual entry
Format: SB##### (e.g., SB12345, SB00860, SB00736)

What Happens Behind the Scenes:

POST /bill/manual_parent_entry Content-Type: application/x-www-form-urlencoded Form Data: - bill_id: [Bill Database ID] - manual_qr: [Parent Bag QR Code]

Business Rules:

  • ‚úÖ Format must be: SB followed by exactly 5 digits
  • ‚úÖ Auto-converts to uppercase (sb12345 ‚Üí SB12345)
  • ‚úÖ Creates parent bag with "pending" status if it doesn't exist
  • ‚ùå Only "completed" parent bags (with 30 child bags) can be attached to bills
  • ‚ùå Cannot attach the same parent bag to multiple bills
