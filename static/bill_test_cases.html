About Your Barcode Scanner

This guide is written for HID Barcode Scanners (the plug-in type that works like a keyboard).

  • Scanner plugs into computer via USB
  • When you scan, the code types automatically into the active field
  • Most scanners add "Enter" key after scanning
  • No special software needed - works like typing

2. SCANNING BAGS TO BILLS

How Scanner Works

Your barcode scanner is a "HID scanner" - it works like a keyboard. When you scan a QR code:

  1. Make sure cursor is in the scan input box
  2. Point scanner at QR code and press trigger
  3. Scanner "types" the code and presses Enter automatically
  4. System processes the scan

4. WEIGHT AND COUNT TESTS

How Weights Work

  • SB bags: Expected 30kg, can hold up to 30 child bags
  • M-format bags: Expected 15kg, can hold up to 15 child bags
  • Actual weight: Number of child bags x 1kg each
